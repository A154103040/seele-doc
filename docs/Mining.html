Mining¶

CPU Mining¶

目前，Seele提供了CPU Miner。

注意，如果您想在Seele挖矿，请务必确保您的Seele Node运行在主网上，且运行的是全节点。若您还不了解如何使用Seele，请阅读Getting Started With seele。

Seele Node启动挖矿¶

您可以在启动Seele Node的同时启动挖矿，Setting Up a Node会帮助您如何构建Node并启动Seele Node。

Seele Node提供了--miner,-m命令行参数来指定Node启动时是否启动挖矿，默认为启动，若不想挖矿可在启动Node时使用--miner stop。Seele Node还提供了--threads命令行参数来设置Node启动的Miner所使用的线程数。

Example¶

Seele Node启动时需要使用配置文件，配置文件的格式参考` <https://github.com/seeleteam/go-seele/wiki/Getting-Started-With-Seele/#node1>`__

// 启动Seele Node时默认启动挖矿

node start -c <configpath>

// 启动Seele Node时启动挖矿，并设置Miner使用的线程数为2

node start -c <configpath> --threads 2

// 启动Seele Node时不启动挖矿

node start -c <configpath> --miner stop

Seele Full Client启动挖矿¶

若您已有了一个Seele Node全节点，您可以使用全节点client来启动并管理您的Node Miner.Create a Full Node Client会帮助您如何获取全节点客户端。

Example¶

Client启动挖矿：

client miner start

true

Client启动挖矿并设置Miner线程数为2：

client miner start --threads 2

true

Client停止挖矿：

client miner stop

true

获取Seele Node Miner运行状态：

client miner status

"Running"

or

"Stopped"

获取Seele Node Miner Coinbase:

client miner getcoinbase

"0x4c10f2cd2159bb432094e3be7e17904c2b4aeb21"

设置Seele Node Miner Coinbase:

client miner setcoinbase "0x4c10f2cd2159bb432094e3be7e17904c2b4aeb21"

true

设置Seele Node Miner线程数为2:

client miner setthreads --threads 2

true

获取Seele Node Miner Engine Information:

client miner getengineinfo

{
    "hashrate": 495812.2433994204,
    "threads": 1
}
